| Key | Action |
|-----|--------|

| **SHIFT** | Toggle Matrix A/B (affects difficulty/coherence) |
| **J** | Activate archetype power (wall jump, phase walk, etc.) |
| **R** | Glitch Pulse (clear nearby hazards) |
| **Q** | Freeze enemies temporarily |
| **C** | Create containment zone |
| **ESC** | Pause menu |

### Shooter Mode (Continuous Action)
Fast-paced combat with real-time movement:

| Control | Action |
|---------|--------|
| **WASD** | Continuous 8-direction movement |
| **Mouse** | Aim direction |
| **Left Click** | Shoot projectiles |
| **ESC** | Pause menu |

### Mode Selection
- Press **M** on the title screen to toggle between Grid and Shooter modes
- More modes coming in future phases (RPG, Constellation, Rhythm, Platformer)
