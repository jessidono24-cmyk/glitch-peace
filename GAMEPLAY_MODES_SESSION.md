## Technical Architecture

### Mode Manager System
```javascript
class ModeManager {
  constructor() {
    this.currentMode = 'grid';
    this.modes = {
      grid: new GridMode(),
      shooter: new ShooterMode(),
      rpg: new RPGMode(),
      constellation: new ConstellationMode(),
      platformer: new PlatformerMode(),
      rhythm: new RhythmMode()
    };
  }
  
  switchMode(modeName) {
    this.modes[this.currentMode].cleanup();
    this.currentMode = modeName;
    this.modes[modeName].init();
  }
  
  update(dt) {
    this.modes[this.currentMode].update(dt);
  }
  
  render(ctx) {
    this.modes[this.currentMode].render(ctx);
  }
}
```

### Mode Interface
All modes implement:
- `init()` - Setup mode
- `cleanup()` - Tear down mode
- `update(dt)` - Game loop
- `render(ctx)` - Draw
- `handleInput(input)` - Input processing
- Access to shared systems (emotional, temporal, learning, etc.)
